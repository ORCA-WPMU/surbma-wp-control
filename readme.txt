== Description ==

Global control plugin for Premium WordPress sites

== Installation ==

1. Upload `surbma-pwp-control` folder to the `/wp-content/plugins/` directory
2. Activate the Surbma - Premium WordPress Control plugin through the 'Plugins' menu in WordPress
3. That's it. :)

== Frequently Asked Questions ==

= What is this plugin good for? =

This plugin was created especially for my clients' websites. It can control some minor, but important parts of a Multisite network, which is very useful if you handle a lot of client websites.

= What does Surbma mean? =

It is the reverse version of my last name. ;)
